# express-template

## ENV
Create .env file and add all the variables ([Example][env-example])

## Development
Install package
```sh
$ npm install
```
Start server
```sh
$ npm run dev
```
Start lint JavaScript
```sh
$ npm run lint
```

## Build
Building bundle.js to directory dist.
```sh
$ npm run build
```

## Production
Start server
```sh
$ npm start
```

[env-example]: https://github.com/GCoder-13/express-template/blob/master/.env.example
